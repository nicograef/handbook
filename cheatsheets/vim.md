# Vim Cheatsheet

| command | description |
| ------- | ----------- |
| :wq     | save and exit |
| yy | copy current line |
| dd | cut/delete current line |
| p | paste into next line |
| :73,81yank | copy lines 73 to 81 |
| :17,39d | delete/cut lines 17 to 39 |
| :%s/_before_/_after_/g | replace _before_ with _after_ in the whole file |
| vim -O file1.ts file2.md | open two files in split screen (vertically) |
| :e _path/to/file_ | open file from within vim |
| :vsplit | split screen vertically |
| Ctrl+w arrow-key | switch between split view files |
